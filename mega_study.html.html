III단원
📦 1. 문제 분해와 모델링
💡 핵심 흐름: 문제 분석 → 추상화(핵심 요소 추출) → 문제 분해 → 알고리즘 설계

📌 문제 분석

정의: 문제를 명확하게 이해하고 주어진 문제 상태와 요구 사항 및 제약 조건을 정의하기 위해 수행

질문의미
현재 상태와 목표 상태는?지금 어떤 상태이고, 어떤 결과가 필요한가
요구 사항은?반드시 해결해야 할 것은 무엇인가
제약 조건은?어떤 한계나 조건이 있는가

🎨 추상화

  • 복잡한 문제를 단순화하는 것
  • 불필요한 요소 제거 → 핵심 요소만 추출
  • 구분 기준은 문제 해결의 목적에 따라 달라짐

✂️ 핵심 요소 추출

  • 추상화의 가장 중요한 단계
  • 문제 해결과 무관한 요소 → 제거
  • 예) 놀이공원 → 주요 놀이시설만 표현

🔪 문제 분해 (4단계)

1
문제를 작은 문제로 분해한다
2
분해한 문제의 해결 가능성 여부를 판단한다
3
해결 가능한 문제로 분해된 경우, 각각 해결한다
4
해결한 결과를 합쳐서 큰 문제 해결로 만든다
III단원
🔧 2. 알고리즘의 이해

📌 알고리즘이란?

  • 문제 해결을 위한 방법이나 절차
  • 문제 해결을 위해 단계별로 정의된 규칙의 집합

🏗️ 알고리즘 제어구조 (★ 필수 암기)

구조설명특징
순차 구조순서대로 명령을 실행위에서 아래로
선택 구조조건에 따라 실행 명령이 달라짐마름모(조건) 기호
반복 구조조건에 따라 특정 명령을 반복 실행루프

✍️ 알고리즘 표현 방법 (3가지)

방법설명특징
자연어형식 없이 말/글로 표현직관적이나 모호할 수 있음
의사 코드프로그램 코드를 흉내 내어 표현정밀함
순서도정해진 기호로 표현시각적으로 흐름 파악 쉬움

📐 순서도 기호 (★ 필수 암기)

기호이름의미
□ (사각형)처리연산이나 데이터 이동 처리
◇ (마름모)조건여러 경로 중 하나 선택
⬭ (둥근 사각형)터미널순서도의 시작과 끝
// (평행사변형)입·출력데이터의 입력과 출력
→ (화살표)흐름선처리의 흐름, 기호 연결
○ (원)연결자흐름이 다른 곳과 연결되는 입구/출구

🔢 정렬 알고리즘

⚡ 효율적인 알고리즘의 조건

  • 작업량이 적고 수행 시간이 짧다
  • 기억 장소(메모리) 사용량이 적다

📊 정렬이란?

데이터를 특정 기준에 따라 순서대로 나열하는 것

오름차순 작은→큰   내림차순 큰→작은

🗂️ 5가지 정렬 알고리즘 (★★ 핵심)

정렬영문핵심 방법
선택 정렬Selection Sort정렬되지 않은 데이터 중 위치에 알맞은 값을 선택하여 이동
삽입 정렬Insertion Sort정렬된 부분의 적절한 위치에 삽입
버블 정렬Bubble Sort이웃해 있는 두 값을 비교·교환하는 것을 반복
합병 정렬Merge Sortn개 서브리스트로 분할 후 각각 정렬하여 합병
퀵 정렬Quick Sort기준값(피벗)을 기준으로 앞/뒤 분류
💡 암기 꿀팁: 선택=선택→이동 | 삽입=삽입 | 버블=이웃끼리 | 합병=분할→합병 | 퀵=피벗

버블 vs 선택 비교 [8,4,2,1]

정렬비교 횟수교환 횟수
버블 정렬총 6회총 6회
선택 정렬총 6회총 2회

→ 선택 정렬이 교환 횟수 면에서 더 효율적!

📦 자료형

분류자료형예시
수치 자료형정수 (int)-3,-1,0,1,2,3
실수 (float)3.14, 5.5
복소수 (complex)3+4j
부울 자료형boolTrue, False
군집(컨테이너)문자열 (str)'hello', "python"

튜플 (tuple)(1,2,3)
집합 (set){1,2,3}
사전 (dict){1:'one'}

🔗 시퀀스 자료형 — 인덱싱·슬라이싱

순서가 있는 자료형: 문자열(' '), 리스트([ ]), 튜플(( ))

구분k='abcdef' 명령문결과
인덱싱print(k[3])d
인덱싱(음수)print(k[-5])b
슬라이싱print(k[1:3])bc
슬라이싱(음수)print(k[-5:-4])b

➕ 연산자 3종류 (★★ 핵심)

종류기호설명
산술 연산자+ - * / // % **// = 몫, % = 나머지, ** = 거듭제곱
비교 연산자> < >= <= == !=True/False 반환. ==같다, !=같지않다
논리 연산자and or not논리곱, 논리합, 부정

🏆 연산자 우선순위 (높은 순)

  1. 괄호 ( ) — 가장 먼저
  2. 거듭제곱 **
  3. 곱셈 * / // %
  4. 덧셈 + -
  5. 비교 연산자
  6. 논리: not → and → or

산술 > 비교 > 논리!
